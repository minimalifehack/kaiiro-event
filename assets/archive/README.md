# 過去開催の写真について

このフォルダに以下のファイル名で写真を置くと、サイトの「これまでの、かいいろまるしぇ」セクションに自動的に表示されます（ファイルが無い場合は、現在のイラスト風プレースホルダーがそのまま表示されます）。

| ファイル名 | 内容 |
|---|---|
| `1st.jpg` | 1st かいいろまるしぇ の写真 |
| `2nd.jpg` | 2nd かいいろまるしぇ の写真 |
| `3rd.jpg` | 3rd かいいろまるしぇ の写真 |

## 推奨

- 比率: 4:3（横長〜正方形に近いもの）。それ以外の比率でも自動でトリミング表示されます
- ファイル形式: `.jpg` または `.png`（`.png` を使う場合は `index.html` 内の `src="assets/archive/1st.jpg"` の拡張子も `.png` に変更してください）
- 容量: あまり大きすぎない（1枚あたり数百KB程度）と表示が軽くなります

## Instagram投稿からの転載について

Instagram投稿の写真・文章を転載いただける旨を確認済みです。投稿に写っている方（出店者・来場者など）への
掲載許諾も併せてご確認の上、ご活用ください。

## キャプションの差し替え

各回の説明文（「はじめての"おかえり"」など）は `index.html` の `<figcaption>` 内、
Instagram投稿文を参考に差し替えてください。
